利用hexo写博客

2016-09-07

安装node

安装hexo

npm install -g hexo

初始化

创建一个用来写博客的目录，并cd到该目录运行:

hexo init
hexo g
npm install hexo-deployer-git --save

配置git

在github.com上面创建.github.io的仓库
在当前目录下，修改_config.yml最下面，修改如下代码并保存
git地址为自己的仓库地址

deploy:
type: git
  repository: git@github.com:dujiecn/dujiecn.github.io.git
  branch: master

发布

hexo d

验证

访问dujiecn.github.io
